# SKILL: Luminar Neo Automation

## Purpose
Allows Ezra to trigger image enhancements and branding overlays using macOS AppleScript.

## Usage
When asked to "Edit a thumbnail" or "Process an image":
1. Move the target image to `~/Documents/Fink_Uploads/raw`.
2. Execute the `scripts/luminar_enhance.scpt` to trigger AI Enhancement.
3. Export the result to `~/Documents/Fink_Uploads/final`.
