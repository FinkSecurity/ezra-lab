# Skill: luminar_basic

**Description:** Use Luminar Neo for photo enhancement and professional editing before final export.

**Current Capabilities (macOS):**
- Open images in Luminar Neo via exec + open command
- Apply presets or AI tools manually (best for high-quality single images)
- Batch export: Use Luminar Neo's built-in batch export (Shift + Cmd + E)
- Recommended workflow: Enhance in Luminar Neo → Export to media/output/ → Further process with FFmpeg/canvas if needed

**Example commands you can run:**
open -a "Luminar Neo" "path/to/image.jpg"

**Future improvement:** If you discover reliable AppleScript automation for Luminar Neo, document the exact script here.
