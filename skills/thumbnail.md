# Skill: thumbnail

**Description:** Generate modern thumbnails suitable for bug bounty write-ups, blog posts, or social media.

**Best Practices:**
- Style: High contrast, bold readable text, modern dark or clean light theme
- Text: Short, impactful title + subtle subtitle
- Output: Always save to media/output/thumbnails/ with name like `topic-thumbnail-style-date-vN.png`
- Variants: Offer dark + light version when useful

**Example usage:**
"Create a dark thumbnail for XSS vulnerability write-up titled 'Reflected XSS Explained'"
